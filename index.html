DIG 245 View Source

Instructions for viewing source code of a webpage in Google Chrome

  1. Open the browser and go to the webpage you want to view the source code of
  2. On that webpage, right-click on the page and click on "Inspect" or "View Page Source"
  3. You can now see the source code for the webpage!

Things you can do with the Dev Tools

  • Debug your code
  • Add Javascript elements to your code in console
  • Check your design on different devices

Metaphor

Using dev tools is like being given the chance to go behind the stage after you watch a magician performs and see all the moving parts of his magic tricks. You get to know how to peform every magic tricks and the equipments needed to accomplish them. You can also learn from the magician and practice all the magic tricks and apply them to your own show.
